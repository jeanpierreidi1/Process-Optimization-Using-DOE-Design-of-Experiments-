# DOE Process Optimization Project

## Overview
This project aims to improve a process by analyzing how different variables impact the outcome using Python, statistical analysis techniques, Excel, and Power BI for data visualization and performance tracking.

## Objective
This project aims to improve a process by analyzing how different factors affect the outcome.

## Tools Used
- Python (Pandas)
- Excel
- Power BI
- Statistical Analysis

## Dataset
The dataset includes:
- Temperature (Low/High)
- Time (Short/Long)
- Material (A/B)
- Quality Score

## Analysis
- Grouped data by variables
- Compared averages
- Identified best performing conditions

## Results
- High temperature improved quality
- Longer processing time increased output
- Material A performed better

## Conclusion
The optimal condition is:
High Temperature + Long Time + Material A
